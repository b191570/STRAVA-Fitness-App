## Strava Fitness EDA Analysis

# Project Overview
This project focuses on Exploratory Data Analysis (EDA) of fitness data from the Strava platform. The goal is to analyze user activity patterns, identify trends, and generate insights that can help improve user engagement and business growth.

# Dataset Description
The project uses multiple datasets:
Daily Activity
Calories
Intensities
Steps
Sleep Data
Weight Logs
These datasets are merged using User ID and Date columns.

# Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Plotly
Scikit-learn

# Data Preprocessing
Merged multiple datasets
Handled missing values
Removed duplicates
Standardized column formats

🔹 Multivariate Analysis

## Key Insights
Strong correlation between steps and distance (0.98)
Active minutes strongly relate to calories burned
Many users maintain consistent step counts
Sleep tracking is relatively low compared to activity tracking
## Business Recommendations
Introduce fitness challenges to boost engagement
Provide rewards/discounts for premium subscriptions
Encourage sleep tracking alongside activity tracking
Improve user awareness campaigns for fitness goals
